# Introdution

- In this React Core Concept mini practice project, I create a mini caculator that caculates the expected return one receives after a duration of time

# Installation

## Requirements

- NodeJS v18 is required beforehand

## Steps

- Run <code>npm i</code> to install all dependencies
- Run <code>npm run dev</code> or <code>npm start</code> to start the development process
